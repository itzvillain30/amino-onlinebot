# amino-onlinebot
a python script for amino you can make unlimited account online any comunity

## Getting Started

### Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/itzvillain30/amino-onlinebot
   cd amino-onlinebot
   ```

2. Execute The Repository In Codespace:

   ```sh
   chmod +x *
   python main.py
   ```
